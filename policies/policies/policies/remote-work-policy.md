# Remote Work Policy

## Purpose
This policy sets out minimum IT and security requirements for employees working remotely.

## Requirements
- Employees must use company-approved devices where possible.
- Devices must have current antivirus protection.
- Devices must have operating system updates installed.
- Employees must use secure passwords and multi-factor authentication.
- Public Wi-Fi should be avoided unless a secure VPN is used.
- Company data must not be stored on unauthorized personal devices or cloud services.

## Support
Employees must report technical issues, lost devices, or suspected breaches immediately.

## Compliance
Remote workers are expected to comply with all company IT and security policies.
