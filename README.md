# IT Governance Templates

## Overview
This repository contains sample IT governance policies, forms, and operational templates for use in business environments.

## Purpose
The purpose of this repository is to demonstrate practical knowledge of IT governance, security controls, operational processes, and documentation standards.

## Included Content
- Acceptable use policy
- Password policy
- Remote work policy
- Backup policy
- Asset allocation form
- Incident report form
- Backup checklist

## Why This Project Matters
IT governance and operational documentation are essential for protecting systems, supporting users, and maintaining accountability in any organization.
